
Résumé sur Git et GitHub :

Git est un système de gestion de versions qui permet de suivre l’évolution d’un projet et d’enregistrer chaque modification. Il est très utilisé pour le travail en groupe et pour garder un historique clair et organisé.

Dans ce TP, j’ai appris à :
- créer un dépôt GitHub ;
- le cloner dans Google Colab ;
- ajouter et modifier des fichiers ;
- faire des commits pour enregistrer les changements ;
- pousser le projet vers GitHub à l’aide d’un token personnel.

Mon idée de projet : créer un petit projet simple et utiliser Git pour suivre ses mises à jour et comprendre le fonctionnement du versionnement.
