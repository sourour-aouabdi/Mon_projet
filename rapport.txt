Résumé sur Git et GitHub :

Git est un système de gestion de versions qui permet de suivre les modifications d’un projet.
Il est utile pour travailler en équipe et pour garder un historique des fichiers.
Dans ce TP, j'ai appris à créer un dépôt GitHub, le cloner dans Colab, ajouter des fichiers et faire des commits.
J’ai aussi appris comment pousser le travail vers GitHub à l’aide d’un token.
Mon idée de projet : créer un petit projet biotechnologique que je pourrai versionner avec Git pour suivre son évolution.
