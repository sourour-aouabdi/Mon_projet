Rapport du TP Git & GitHub
Aouabdi Sourour
Aggouni Hibat Allah
Spécialité : Toxicologie fondamentale appliquée

Résumé :
Git est un outil très utile car il permet de suivre l’évolution d’un projet, d’enregistrer toutes les versions et de revenir à n’importe quel état précédent en cas d’erreur. Dans ce TP, j’ai appris à créer un dépôt Git dans Google Colab, configurer mon identité, ajouter des fichiers, créer des commits et envoyer le projet sur GitHub grâce à git push. J’ai également appris à relier un dépôt local à un dépôt distant à l’aide de git remote add origin.

Projet personnel :
Je souhaite développer un petit projet biotechnologique permettant d’enregistrer et de suivre des données d’expériences de laboratoire. Git me permettra de conserver un historique clair, d’organiser les modifications et de travailler correctement sur différentes versions du projet.

Branche develop :
J’ai créé une branche de développement avec la commande : git checkout -b develop.
